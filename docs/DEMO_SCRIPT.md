# MasterFlow three-minute demo

## 0:00-0:20 — frame the problem

“Master Electronics already has team queues and specialized request forms, but employees must search a very large catalog to find the right one. MasterFlow keeps the routing and business requirements while replacing category hunting with one simple request experience. It also creates a controlled workspace for freight opportunities before cost is incurred.”

## 0:20-1:05 — Smart Request

1. Start in **Regular user** view on `index.html`.
2. Enter: `I need ink for the Zebra printer at Pack Station 14`.
3. Show that MasterFlow selected **Printer Ink Request**, explained the match, displayed confidence, and prefilled printer, location, ink type, and requester.
4. Point out that only template-required information is shown and the employee can choose another request type.
5. Submit, show the numbered confirmation, then open **My Requests** and find the new record.

## 1:05-1:35 — warehouse P1 fast lane

1. Select **Shipping is stopped**.
2. Explain that this path never waits for AI classification.
3. Enter fictional location, process, start time, affected stations, and symptom.
4. Submit, switch to **Ticket receiver**, and show the P1 in **Warehouse Systems / On-call** with its SLA.

## 1:35-2:20 — freight optimization

1. Open **Freight Optimization**.
2. State: “Every eligible order can be scanned, but only actionable exceptions create work.”
3. Open **ABI173 / ABCD14-02**.
4. Show current freight, proposed freight, $543.54 internal savings, $330.56 customer savings, 67% confidence, related backlog, one-day promise-date impact, decision cutoff, automatic release, and failed guardrails.
5. Send the draft to Sales or record a controlled release/hold decision. Clarify that no live customer email or ERP update occurs.

## 2:20-2:45 — no-code administration

1. Switch to **Administrator**.
2. Open **Request templates**.
3. Change a queue, SLA, trigger phrase, or required field and save.
4. Return to Home and show that Smart Request immediately uses the new configuration.

## 2:45-3:00 — close

“MasterFlow reduces friction for every employee, preserves business ownership, protects warehouse-critical response, and gives authorized teams an auditable way to pursue freight savings. The prototype proves the workflow; production integrations would follow behind approved security and governance.”
